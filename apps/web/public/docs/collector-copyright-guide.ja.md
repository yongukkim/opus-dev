# 所蔵者のための著作権・共有ガイド

本ガイドは、**コレクションとしての体験と記録の価値をご自身で守る**ための整理です。**法務レビュー前の一般案内ドラフト**であり、[著作権・利用について](/legal/copyright)および[利用規約](/terms)とあわせてご覧ください。

---

## 希少性と市場価値

限定エディションの**希少性**とそれに伴う**市場価値**は、多くの所蔵者にとって誇りの源泉になります。一方で、無断複製・無断共有・商用の再利用はその印象を損ない、作家・プラットフォームが定める利用範囲と抵触し得ます。**公開チャネルではウォーターマーク付き・低解像度の派生画像**を用い、**原本・高解像データ**は OPUS が提供する経路（マイページ、認証済みスマートフォン・タブレットのモバイルWebで許可された鑑賞画面など）内でのみご利用いただくことが、記録と信頼をともに守る助けになります。

---

## 高解像鑑賞の想定デバイス

OPUS が提供する**認証済みの高解像・タイル鑑賞**は、**スマートフォン・タブレットのモバイルWeb**を前提に設計されています。**印刷物への出力**や**大型の外部モニター・テレビ・プロジェクター等**を想定した再生・出力は、本サービスが担う範囲に含まれず、その目的での無断複製・送信は著作権法上の問題となり得ます。

## 推奨する利用 / 避けたい利用

| 推奨（DO） | 避ける（DON'T） |
| --- | --- |
| SNS・ブログなど**公開**にはウォーターマーク付き・低解像度の**安全な派生**のみ | サービス外への**原本・高解像ファイル**の無断配布・再ホスティング |
| OPUS **マイページ**と規約に沿った機能で鑑賞・所蔵記録の確認 | 技術的保護手段の回避、**出所・エディション表示の歪曲** |
| 認証済み**スマートフォン・タブレット**のモバイルWebで許可された高解像鑑賞 | **印刷**・**大型モニター・テレビ等**への無断出力・公衆送信 |
| 侵害の疑いは **admin@opus-store.com** へ | 他人の著作物を**自分のもののように**掲載したり商用利用したりすること |

## 法的な文脈（日本）

無断の**複製・頒布・送信可能化**などは、**日本の著作権法等**に基づき、民事上の差止・損害賠償、刑事罰の対象となり得る**厳しい法的措置**が講じられ得ます。適用の有無は事案・管轄により異なり、本文は**法的助言ではなく**、**OPUS が当事者に代わって執行・訴訟を行う旨を意味するものではありません。**

## The Chronicle・認定書との関係

The Chronicle とエディション認定書は、**鑑賞・所蔵・来歴を示す記録**を補助します。それ自体が第三者に対する**執行力のある権原**を自動的に付与するものではなく、著作権・利用範囲は**作家・規約・著作権案内**が優先します。
